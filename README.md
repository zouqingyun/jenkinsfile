# 环境说明

| 环境简称   | 环境全称                            | 说明                                                 |
| -------- | ----------------------------------- | ---------------------------------------------------- |
| DEV      | Development environment             | 开发环境，用于开发者调试使用                              |
| FAT      | Feature Acceptance Test environment | 功能验收测试环境，用于软件测试者测试使用                    |
| UAT      | User Acceptance Test environment    | 用户验收测试环境，用于生产环境下的软件测试者测试使用          |
| PRO      | Production environment              | 生产环境                                              |
